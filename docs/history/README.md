# CXF Publication Archive and Development History

This folder contains the public-facing English publication set prepared from the supplied CXF draft series and the Version 1.0 release material.

## Contents

- `CXF_Development_History_Draft_0.1_to_1.0_EN.pdf`  
  Narrative history of the draft series from 0.1 through 0.19 and the release of Version 1.0.
- `CXF_Version_1.0_Release_and_Consensus_Edition_EN.pdf`  
  Polished public release rationale and consensus document for Version 1.0.
- `CXF_Version_1.0_Executive_Summary_EN.pdf`  
  Short overview of the 1.0 release scope and rationale.


- `drafts/README.md`  
  Separate index of the individual pre-release draft archive entries from 0.1 through 0.19.

## Separate draft archive

For readers who want each iteration to be visible individually, a separate draft archive is available under [`drafts/`](drafts/README.md).
Each entry is presented as a normalized English archival page suitable for public repository publication.

## Editorial note

This publication set was normalized for public repository use:

- English language throughout.
- Consistent layout and presentation.
- Personal names removed or replaced with `h4shire`.
- Intended as repository publication material, not as a replacement for the normative spec files under `spec/`.

## Recommended repository integration

Recommended location: keep this archive under `docs/history/` so readers can clearly distinguish between:

- the **normative specification and schemas**,
- the **user-facing documentation**, and
- the **historical publication archive**.
